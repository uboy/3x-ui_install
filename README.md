# 3x-ui install

## Назначение
Репозиторий содержит скрипт для автоматической установки и базовой настройки 3x-ui на Ubuntu.

## Состав файлов
- `install-3xui.sh` — основной установочный скрипт.
- `.gitignore` — исключения для временных файлов, артефактов и секретов.
- `README.md` — краткая инструкция по запуску.
- `DESIGN.md` — дизайн и решения по установщику.
- `PLAN.md` — план реализации.

## Быстрый старт
```bash
chmod +x install-3xui.sh
sudo bash install-3xui.sh
```

## Важно
Скрипт выводит учетные данные (логины/пароли/секреты) в консоль. Сохраните их в безопасном месте и ограничьте доступ к истории терминала.
